Why LaTeX?

Unlike "What You See Is What You Get" (WYSIWYG) editors like Microsoft Word, LaTeX uses a markup language approach. This means you focus on the content and structure of your writing, while LaTeX handles the formatting and typography with mathematical precision.

Join us for a hands-on workshop where we demystify LaTeX, the gold standard for high-quality document preparation used by scientists, engineers, and researchers worldwide.

What You Will Learn

This workshop covers the essential building blocks to get you up and running:

  • The Big Picture: Understanding the TeX ecosystem and setting up your environment.
  • Document Anatomy: Mastering the preamble, document classes, and the \begin{document} environment.
  • Choosing Your Format: How to structure different document types including Articles, Books, and Reports.
  • Typesetting Basics: Managing fonts, lists, sections, and special characters.
  • Math Mode: Creating beautiful equations and formulas that look perfect every time.
  • Graphics & Tables: How to insert images and build clean and professional tables.
  • Automated Citations: A brief introduction to managing bibliographies with BibTeX.
  • Dynamic Presentations: Using Beamer to create stunning slideshows and academic posters.

Getting Ready: Setup & Distributions

To get the most out of this workshop, please have a working LaTeX environment on your laptop. You have two main options:

Option A: The Cloud (Easiest)

Create a free account at Overleaf.com. No installation is required, and you can access your work from any browser.

Option B: Local Installation (Offline Use)

If you prefer to work offline, install a TeX distribution based on your operating system:

  • Windows: Download MiKTeX or TeX Live.
  • Debian / Ubuntu / Mint: Use sudo apt install texlive-full.
  • Red Hat / Fedora / AlmaLinux: Use sudo dnf install texlive-scheme-full.
  • macOS: Download MacTeX.
Technical Tip:
Installing a full distribution (like texlive-full) ensures you have all necessary packages for the workshop without needing an internet connection during the sessions. These installations are large (~5GB), so please download and install them before arriving at the venue!
